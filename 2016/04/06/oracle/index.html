oracle

oracle
  • oracle

视图

  1. 简化查询
  2. 控制数据访问
  3. 数据独立性
  4. 避免访问重复的数据

结构：
create [or replace] [force|noforce] view view1 [(alias[, alias]…)] as subquery;

描述视图的结构
describe view1;
删除视图
drop view view1;
临时视图

临时视图是嵌套在sql语句中，本身不会存储。

Top-N
select [column_list],rownum
from (select [column_list]
from table
order by top-n_column desc)
where rownum<=n;

序列

  • 给主键提供值
  • 共享对象
  • 自动提供唯一的数值

语法：
create sequence sequence1
[increment by n]//公差是多少
[start with n]
[{maxvalue n | nomaxvalue}]
[{minvalue n | nominvalue}]
[{cycle n | nocycle}]
[cache n | nocache];

currval 中存放序列的当前值
nextval 返回序列中的下一个有效值
使用序列访问效率提高

索引

优点：

  • 一种数据库的对象
  • 指针加速oracle服务器的查询速度
  • 快速定位数据的方法，减少磁盘i/o
  • 索引与表相互独立
  • oracle服务器自动使用和维护索引

缺点：

  • DML操作的更新复杂

创建索引
create index index1 on table (column[, column]…);
create index emp_last_name_idx on employees(last_name);
不要创建索引：

  • 表很小
  • 列不经常作为连接条件或出现

  • 经常更新

    同义词

    好处：

    方便其他用户的访问
    缩短对象名的长度

    CREATE [PUBLIC] SYNONYM synonym FOR object;

    权限

    要点：

  • 创建用户
  • 创建角色
  • 使用grant和revoke语句赋予和回收权限
  • 创建按数据库联接
