web-分页

jsp
  • javaweb
  • jsp

分页实现

  1. 页面中提供分页参数
  2. 后台类中实现分页查询

页面中提供的分页参数

  1. 目标页码
  2. 每页多少数据（行）
  3. 一般还要有共多少页或多少数据

后台类中实现分页查询

  1. 根据页码传递的分页参数计算查询记录的起始位置和结束位置
  2. 将计算的起至位置带入sql代码中去作为查询条件

数据库不同分页查询的sql代码

  1. SQLserver 使用关键字top
  2. ORACLE 使用rownum关键字
  3. MYSQL 使用limit关键字实现分页目的
